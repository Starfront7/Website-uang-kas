# Uang Kas Kelas - PHP + MySQL + Bootstrap (Complete project)
This is a ready-to-run simple management system for class funds (uang kas) with:
- Login (admin)
- Dashboard with totals and charts
- Siswa (students) CRUD
- Kas Masuk (income/payments) CRUD
- Kas Keluar (expenses) CRUD
- Simple reports and Chart.js integration

## Setup
1. Put this project into your local webserver folder (e.g., `htdocs` or `www`).
2. Create a MySQL database named `db_uangkas` and import `db_uangkas.sql`.
3. Adjust `koneksi.php` if your MySQL user/password differ.
4. Open `http://localhost/uangkas_project/login.php`

Default admin credentials:
- username: `admin`
- password: `admin123`
